# Support

CodexIAChat is currently an early open-source documentation project.

## Questions

Use GitHub Discussions if enabled, or open an issue with the `question` label.

## Security Issues

Do not open a public issue for secrets, private infrastructure exposure, or vulnerabilities. Follow `SECURITY.md`.

## Implementation Help

When asking for implementation help, include:

- the exact goal;
- the files or documents involved;
- the runtime assumptions;
- any command output relevant to the failure;
- what has already been tested.
